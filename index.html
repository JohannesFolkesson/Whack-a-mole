Spelinstruktioner

Mål: Whacka så många mullvadar som möjligt innan tiden tar slut!

Hur man spelar:

  • Klicka på Start för att börja spelet
  • Mullvadar dyker upp slumpmässigt på spelbrädet
  • Klicka snabbt på mullvadarna för att få poäng
  • Du har 60 sekunder på dig
  • Klicka på Reset för att börja om

Poäng: Varje träff ger dig poäng. Försök slå ditt rekord!
